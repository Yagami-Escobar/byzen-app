  •   Extensiones de Pestañas.
  •  Perfilado.
  •  Diseñado.
  •  Podrás elegir entre 3 tamaños Small, Medio o Large.

Procedimiento:

  •  Se prepara las propias pestañas para que se puedan colocar correctamente las extensiones. Tras limpiarlas bien, se coloca un parche para proteger las pestañas inferiores.
  •  Se procede a pegar las extensiones sobre las pestañas colocando una a una.
  •  A medida que se avanza, se va peinando las extensiones, así estarán desenredadas.
  •  Las extensiones de pestañas no se verán postizas, no dejan grumos, no se visualiza el pegamento.
